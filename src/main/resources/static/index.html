API Documentation

This is a RESTful API for managing To-Do tasks. All endpoints return JSON data.

Task Object Structure

{ "id": 1, "title": "Task title", "description": "Task description", "status": "todo", "createdAt": "2024-01-01T10:00:00", "updatedAt": "2024-01-01T11:00:00" }

Status Values

  • todo: Task is pending
  • in_progress: Task is in progress
  • done: Task is completed

Common Errors

  • 404: Resource not found
  • 400: Bad request (invalid input)
  • 500: Internal server error

Testing Tools

You can also test the API using:

  • Swagger UI: /swagger-ui.html (if configured)
  • Database Console: /h2-console (if configured, JDBC URL: jdbc:h2:mem:todo_db)
  • cURL: Use the command line for testing
  • Postman: Import API collection
